Excursions in Multiplicative Number Theory:

Typos




The typos of the book "Excursions in Multiplicative Number Theory" are collected here.

Thanks are due to J. Avigdor from Charenton Le Pont and to Allen Stenger. (Last update: July 25th, 2023)
  1. P. x, Acknowledgment line 1.
    french should be French.
  2. P. 6, line 5 of section 1.3.
    The last element in the list should be $p^\alpha$, not $p^{\alpha - 1}$
  3. P. 8, Exercise 1.12.
    The question should be:
    Show that we have $\varphi(n)\ge \bigl($ ${\frac{2}{9}}$ ${}\bigr)^{{1}/{3}}n^{{2}/{3}}$ for all $n\ge 1$.
    A $\frac{9}{2}$ took the place of the $\frac{2}{9}$ in the book.
  4. P. 26, line 3.

  6. P. 42, line 5 from bottom.
    ``On using $1 + \alpha_p \le \log(2n)$'': This inequality is true when $p=2$, where it should be "$1 + \alpha_2 \le \log(2n)/\log 2$". In that case $\log(1+\alpha_2)\le \log\log(2n)-\log\log 2\le (1-\frac{\log\log 2}{\log\log 6})\log\log(2n) \le 1.7 \log\log(2n)$. To correct the proof, set $u_2=1.7$ and $u_p=1$ when $p\ge 3$. We have $$ \sum_{p^{\alpha_p}\|n \\ p < P}\log(1+\alpha_p) \le \log \log(2n)\sum_{p < P} u_p \le P \log \log(2n). $$ The proof then proceeds as before.
