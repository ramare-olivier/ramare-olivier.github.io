PARIEMACS: GP-SCRIPT-MODE / GP-MODE




Dernière mise à jour : 14 janvier 2022.
Latest update : January 14th 2022.

Cette page héberge les dernières versions des modes dédiés à l'utilisation conjointe de emacs et de pari/gp. Il s'agit d'une combinaison de deux modes, l'un gp-script-mode pour editer des scripts pour gp, et l'autre gp-mode pour exécuter gp, tous les deux avec l'aide de l'éditeur emacs. La combinaison de ces deux modes s'appelle PARIEMACS, et anciennement pari-mode, ce qui était assez mal choisi.
This page is dedicated to two emacs modes written to use jointly emacs and pari/gp. This is in fact a junction of two modes: gp-script-mode to edit scripts for gp, and gp-mode to run gp, both with emacs as an editor. This combination is now called PARIEMACS though the (poorly chosen) name pari-mode is still sometimes used.
