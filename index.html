 Scripts GP :
-- en libre accès même si ce ne sont pas des merveilles --
  • [1] L'article "Explicit estimates on several summatory functions involving the Moebius function" nécessite quelques scripts qui sont accessibles ici :
    1.   [1-1] CompIntM.gp: this script computes the integral of $\max(0,M(x))$ for $x$ between 1 and a given bound, where $M(x)$ is the summatory function of the Moebius function.
    2.   [1-2] AsymptoticBoundsFor_m.gp: this script checks bounds like $|\sum_{n\le x}\mu(n)/n | \le 1/\log(x)$ for $x$ in some finite range. The RHS can be modified at will.
    3.   [1-3] AsymptoticBoundsFor_checkm.gp: this script checks bounds like $|\sum_{n\le x}\mu(n)\log(x/n)/n | \le 1/\log(x)$ for $x$ in some finite range. The RHS can be modified at will.
    4.   [1-4] AsymptoticBoundsFor_checkcheckm.gp: this script checks bounds like $|\sum_{n\le x}\mu(n)\log^2(x/n)/n | \le 1/\log(x)$ for $x$ in some finite range. The RHS can be modified at will.
    5.   [1-5] AsymptoticBoundsFor_M.gp: this script checks bounds like $|\sum_{n\le x}\mu(n) | \le x/\log(x)$ for $x$ in some finite range. The RHS can be modified at will.
    6.   [1-6] AsymptoticBoundsFor_mstar.gp: this script checks bounds like $|m^*(x)| \le 1/\log(x)$ for $x$ in some finite range, where the function $m^*(x)$ is defined in equation (7.1). The RHS can be modified at will.
    7.   [1-7] AsymptoticBoundsFor_checkmstar.gp: this script checks bounds like $|\check{m}^*(x)| \le 1/\log(x)$ for $x$ in some finite range, where the function $\check{m}^*(x)$ is defined in equation (10.1). The RHS can be modified at will.
    8.   [1-8] AsymptoticBoundsFor_checkcheckmstar.gp: this script checks bounds like $|\check{\check{m}}^*(x)| \le 1/\log(x)$ for $x$ in some finite range, where the function $\check{\check{m}}^*(x)$ is defined in equation (13.1). The RHS can be modified at will.
    9.   [1-9] AsymptoticBoundsFor_Mstar.gp: this script checks bounds like $|M^*(x)| \le x/\log(x)$ for $x$ in some finite range, where the function $M^*(x)$ is defined in equation (16.5). The RHS can be modified at will.
